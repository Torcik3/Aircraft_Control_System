# Aircraft Attitude Control System

Nonlinear UAV attitude control system implemented in **MATLAB/Simulink**.
Comparison between classical **PID (CHR tuned)** and **Mamdani Fuzzy Logic Controller**.

### Key Features:
* **Euler rotational dynamics** implementation.
* **Inertia coupling** analysis.
* **PID tuning** using the Chien-Hrones-Reswick (CHR) method.
* **Fuzzy PD controller** design.
* Various **simulation scenarios**.

### Requirements:
* MATLAB R2023a or newer.
* Simulink, Fuzzy Logic Toolbox.

### How to run:
1. Open `controlSpatialOrientation.slx`.
2. Run the simulation.
3. Check the results in the Scope/Workspace.
